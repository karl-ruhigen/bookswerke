# Говорящие роботы, андроиды и разумные компьютеры - как пример "странных людей"

В большинстве книг, фильмов и игр про будущее обязательно есть разымные, говорящие, работящие роботы. Пошло это скорее всего со знаменитой пьесы Чапека [https://ru.wikipedia.org/wiki/R.U.R.], в которой был введен новый персонаж - "робот".
